# STUDENT-RECORD-MANAGEMENT
Student Record Management System

A simple Student Record Management System built with Python to manage student details such as adding, viewing, updating, searching, and deleting records. This project is ideal for beginners learning Python concepts like file handling, functions, and CRUD operations.


Features


Add new student records
View all student records
Search student by ID
Update student information
Delete student records
Store data using file handling / database
Simple command-line interface


Technologies Used

Python 3
File Handling / SQLite (depending on your project)
VS Code / PyCharm
